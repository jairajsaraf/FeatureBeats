# Data Directory

This directory contains the Spotify dataset and processed data.

## Structure

- `raw/`: Original dataset files (not tracked in git)
  - Place `data.csv` from Kaggle here: https://www.kaggle.com/datasets/ektanegi/spotifydata-19212020
- `processed/`: Cleaned and processed data files
- `external/`: Any additional external datasets

## Dataset

Download the Spotify dataset (1921-2020) from:
https://www.kaggle.com/datasets/ektanegi/spotifydata-19212020

Place the `data.csv` file in the `raw/` directory.

**Note:** Large data files are not tracked in git. Use the provided scripts to download and process the data.
